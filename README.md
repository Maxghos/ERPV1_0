# Cercotec ERP

Proyecto base creado con Vite + React + TypeScript + Tailwind CSS.

Pasos para ejecutar:

1. Instalar dependencias:

```bash
npm install
```

2. Levantar servidor de desarrollo:

```bash
npm run dev
```

Estructura principal:

- src/: código fuente
- src/components: Layout y Sidebar
- src/pages: Dashboard, Cobranza, Inventario, Ventas, Clientes
- src/context: Context API global
